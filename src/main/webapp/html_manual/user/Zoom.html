Zoom


Las siguientes funciones de zoom no permanecen activas, después del clic sus acciones no se detienen.


Herramientas de Navegacion


  • Zoom Más: Permite acercarse usando el centro del mapa como punto de enfoque
  • Zoom Menos: Permite alejarse usando el centro del mapa como punto de enfoque
  • Zoom Extensión: Permite visualizar una vista con la extensión nacional (Republica de Honduras)
  • Zoom Anterior: Permite regresar a una vista del mapa posterior al actual.
  • Zoom Posterior: Permite avanzar a una vista del mapa posterior al actual.
  • Zoom Coordenadas: Permite uno zoom sobre el punto correspondiente a las coordenadas X- Y insertadas. El sistema de referencia del mapa está configurado por el Administrador a través del Panel de Administración. (Predefinido: WGS84 UTM/16 N)
