Barra de Herramientas Extendidas


Permite varias acciones específicas, algunas de las cuales pueden necesitar autorización.


Todas las funciones de la Barra Extendida permanecen activas cuando pulsadas (se tornen azul). Haciendo un segundo clic se desactivan.


Las herramientas disponibles son:


    1. Impresión
    2. Búsqueda de datos
    3. Búsqueda de metadatos
    4. Anadir capa de WMS externos
    5. Cargar KML
    6. Descargar una capa
    7. Activar los dibujos
    8. Activar los Reportes de Incidencia
