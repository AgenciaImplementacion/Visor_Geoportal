Medida de áreas


Para medir áreas:

  1. Activar la herramienta de medida de áreas
  2. Con el ratón, dibujar el polígono que se quiere medir. Para terminar la medida, hacer doble clic
  3. La medida aparece en la barra abajo (Punto 3), en hectáreas y en km^2
