Configuración de usuarios


Permiten la administración de los usuarios, y la asociación con un rol definido.


Herramientas de esta sección  permiten: Crear, Eliminar, Modificar y Activar/Desactivar usuarios.




La creación de un usuario necesita las siguientes informaciones:

          • Nombre
          • Contraseña
          • Rol
